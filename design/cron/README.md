# 定时任务（Cron）模块

## 模块概述

定时任务模块是系统的独立核心模块，负责任务的定时调度、执行和管理。它与工作流模块是协作关系，而非包含关系。

## 核心职责

| 职责 | 说明 |
|------|------|
| **任务调度** | 支持多种调度策略（at/every/cron） |
| **任务执行** | 触发 Agent 或工作流执行 |
| **执行历史** | 记录每次任务的执行结果 |
| **调度器管理** | 调度器状态监控和配置 |
